# Portfolio React (Vite)

Projet React minimal créé par l'assistant. Contient pages: Home, About, Services, Work, Blog, Contact, case studies.

Pour démarrer:

1. Installer les dépendances

```bash
npm install
```

2. Lancer le serveur en développement

```bash
npm run dev
```

Le site sera accessible sur l'URL indiquée par Vite (par défaut http://localhost:5173).
